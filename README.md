# Tabs System ✨

## 🌐 Live Demo

👉 https://johnyisbackk.github.io/js-tabs-system/

---

Modern **tabs UI component** built with pure HTML, CSS, and vanilla JavaScript.

This project focuses on clean state switching, data attributes, and dynamic content display.

---

## 🚀 Features

* 🧩 Reusable tabs component
* 🎯 Active tab switching
* 📄 Dynamic content panels
* 🧠 Data attribute mapping
* ⚡ Smooth UI state changes
* 💎 Clean modern design

---

## 🎮 How It Works

1. Click a tab button
2. The clicked tab becomes active
3. Previous active tab is removed
4. Matching content panel is displayed
5. Other panels are hidden

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript
